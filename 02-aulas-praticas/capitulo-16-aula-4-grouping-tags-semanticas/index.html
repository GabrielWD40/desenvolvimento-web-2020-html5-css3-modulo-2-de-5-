BEM VINDO AO MEU SITE

link link link link link

Antigamente, tudo se baseava nas DIV´s, sempre que você queria fazer um menu, um rodapé ou então uma área principal era usado o tag div. Atualmente, isso foi mudado. Agora existem groups tags semânticas, que ajudam os mecanismos de buscas do seu site.

Nessa aula nós vimos as tags:

  • HEADER → cabeçalho
  • MAIN → parte principal do seu site
  • FOOTER → rodapé do seu site
  • SECTION → Usada como se você uma div, onde indica o início de uma coisa e o fim dela
  • ARTICLE → Usada quando há elemento de textos
  • ASIDE → Conteúdo relativo ao artigo
