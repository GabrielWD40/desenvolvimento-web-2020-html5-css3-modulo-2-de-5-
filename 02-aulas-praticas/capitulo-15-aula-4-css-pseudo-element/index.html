Mais atributos das pseudoclasses e pseudoelementos

Nessa aula vamos ver mais algumas coisas sobre as pseudoclasses, como os atributos

  • visited: quando um link é visitado. É o estilo que ele ficará.
  • Active: A cor de quando clicamos em alguma coisa.

As pseudoclasses possuem apenas um " : " já os pseudo elementos possuem " :: " dois, dois pontos.

Isso significa que ali é um pseudoclasse para elementos.


Nas pseudoclasses para elementos temos o ::BEFORE e ::AFERTER, que vão indicar o que vem antes de um elemento e o que vem depois.
