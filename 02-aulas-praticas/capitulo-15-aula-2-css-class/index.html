DIFRENÇAS ENTRE ID e CLASS: Dentro de um documento HTML é possível ter apenas um elemento ID.
VOCÊ SÓ PODE USAR ELE UMA ÚNICA VEZ.

Se você deseja aplicar uma estilização em mais de um elemento ao mesmo tempo, use o class="nome_da_classe"

Além disso, é possível ter várias classes ao longo do seu corpo HTML.

Para referenciar uma classe no CSS, use .nome_da_class{}
Outra coisa super legal é poder usar duas class num mesmo elemento, como nos exemplos abaixo. Detalhe: o separamento é feito por espaços.


VOCÊ TAMBÉM CONSEGUE COLOCAR UM ELEMENTO COM ID e CLASS. O ID sobrepõem a class.
