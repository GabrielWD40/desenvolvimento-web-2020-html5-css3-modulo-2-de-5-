Nessa aula, nós vamos aprender a fazer uma borda a partir de uma imagem. Para isso, será necessário algumas coisas:

  1. Ter uma imagem a qual se deseja adicionar uma borda
  2. Aplicar a estilização borda dentro de um elemento, caso contrário, não funcionará.
  3. border-image: url('caminho da sua imagem')
  4. border-image-slice: Usado para fatiar uma imagem
  5. border-image-reapet: para a imagem se repetir
  6. border-image-reapet: stretch → Vai puxar a linha, esticando-a como no primeiro exemplo!

Você pode usar um shorhand para simplificar os código abaixo:

    
    
        H1#T3{
            border: 15px solid black;
            text-align: center;
            border-image: url('borda.png');
            border-image-slice: 28;
            border-image-repeat: repeat;
        }

ao invés disso, use o border-image na mesma sequência acima: 

    
        border-image: url('nome do arquivo') 28 repeat

exemplo: 

        
            H4#T4{
                border: 15px solid transparent;
                text-align: center;
                border-image: url('borda.png') 28 repeat;
            }
