CRIANDO VARIÁVEIS COM CSS

Para criar uma variável em CSS, é necessário declará-las na raiz do seu código, para isso use: 

                    
                        :root {
                            --nome_da_sua_variável: #83e1ad;
                            --cor1: #c5ebd6;
                            --cor2: #83e1ad;
                            --cor3: #3ddc84;
                            --cor4: #2fa866;
                            --cor5: #1a5c37;
                            --cor6: #063d1e;
                        }

Essa raiz é como a área de declaração global das variáveis em outras linguagens de programação, como C, Java ou python!

Como usar?

Para usar as variáveis com as cores é muito fácil, basta informar o nome delas na estilização. Exemplo:

texto 1

texto 2

texto 3

texto 4

texto 5

texto 6


                    
                    <p> style="color: var(--cor1);">texto 1</p>
                    <p> style="color: var(--cor2);">texto 2</p>
                    <p> style="color: var(--cor3);">texto 3</p>
                    <p> style="color: var(--cor4);">texto 4</p>
                    <p> style="color: var(--cor5);">texto 5</p>
                    <p> style="color: var(--cor6);">texto 6</p>

Aperte control + espaço e role até o final, depois escolha a opção desejada!
