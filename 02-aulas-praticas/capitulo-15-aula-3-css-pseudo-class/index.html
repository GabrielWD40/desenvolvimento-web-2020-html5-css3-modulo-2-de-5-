CSS pseudo-class

Nessa aula acabamos vendo um pouco sobre as DIV´s que ocupam uma linha toda dentro do HTML. Vimos tag chamada de display; inline-block → que coloca tudo na mesma linha e vai quebrando para baixo conforme o espaço diminui

Vimos que as classes são feitas como .nome_da_classe{}, já as pseudoclasses são feitas por ":". Elas precisam estar relacionadas a um elemento ou uma classe. Elas tratam de um estado de um elemento.

  • elemento :HOVER → Vai tratar do elemento quando o mouse passar por cima.

Outra coisa muito legal que vimos na aula foi a questão de um texto estar escondido dentro de uma DIV.

Primeiro de tudo, nós informamos no cabeçalho style que há um elemento p escoldido através de:

            nome_da_div > p {
                display: none;
            }

Depois disso, precisaremos fazer uma regra pro código acima.

            nome_da_div > p: hover{
                estilização que você deseja 
            }
