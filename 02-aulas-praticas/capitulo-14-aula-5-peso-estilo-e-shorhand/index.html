O elemento que cuida do peso de uma fonte se chama FONT-WEIGHT (peso da fonte).

Existem três tipos:

  1. lighter
  2. normal
  3. bold
  4. bolder

Outra estilização que existe é a font-style: italic., que vai deixar o seu texto em itálico.
Text decoration: underline. → Vai deixar o seu texto com um sublinhado embaixo.
ATENÇÃO: Você tem que ver se a sua fonte permite as estilizações citadas acima. Porque senão, as alterações feitas não irão funcionar.

< style="text-decoration: underline;">Texto com underline >

< style="font-weight: bold;">Texto com peso >

< style="font-style: italic;" >

Uma coisa muito legal que dá para ser feita é simplificiar todos esses códigos acima usando o FONT:!

Os parâmetros são:

  1. font - style (estilos na fonte, como underlines)
  2. font - weight (peso de fonte)
  3. font - size (que é o tamanho da fonte)
  4. font - family (que é a família da fonte)

Chamamos essa propriedade de shorhand, que é uma "mãozinha... Ela permite simplificar tudo num código só."
