Seletores personalizados

Nessa aulas, vamos criar seletores personalizados. Isso é importante quando queremos estilizar dois ou mais elemenentos da mesma classe com estilos diferentes um do outro.

Para isso, nós damos uma identificação para o nosso elemento, o que chamamos de ID="nome_dado".
Para referenciar esse elemento no CSS, use tag_do_elemento#nome_da_id.
Exemplo: " h1#principal{} " ou
Exemplo: " #principal{} "


O que for ID em HTML, em css vira '#'
O que for class em CSS, em CSS vira '.'
